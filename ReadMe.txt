C.A.M.P.アーセナルスクリプト生成ツール
by towa ft. size-chura

質問、意見、不具合等あればtowaまで連絡下さい。


// 概要
兵科別アーセナルスクリプトを生成するツールです。



// 使い方
1. 全兵科共通の装備を追加。
   画面上部「追加」ボタン横のテキストボックスに追加したい装備のクラス名を記載し、「追加」ボタンをクリック（またはEnterキー）
   誤って追加したものは「選択を除去」、「全て除去」ボタンを利用して削除可能。


2. 兵科別の装備を追加。
   画面下部「兵科追加」ボタンをクリックすることで、新規の兵科タブを追加可能。
   「兵科名」テキストボックスに兵科名を入力する。
   兵科タブ内の「追加」ボタン横のテキストボックスに追加したい装備のクラス名を記載し、「追加」ボタンをクリック（またはEnterキー）
   誤って追加したものは「選択を除去」、「全て除去」ボタンを利用して削除可能。


3. 「スクリプト出力」ボタンを押してsqfファイルを生成。
   Armaミッションへの適用方法は、出力されたsqfファイル内のコメントを参照。


※ 各装備のクラス名は、Arma3内AceArsenalにて装備選択後、「CTRL」と「C」キーを同時押しすることで、コピー可能。
   コピーした状態で、テキストボックスに張り付けて「追加」ボタンを押すことで指定装備が追加できる。



// その他機能
・「最低限装備」ボタンを押すと、exeフォルダ内Minimum.txtを読み込み、医療品やグレネード等装備を追加できる。
   Minimum.txtを編集することで、読み込む装備編集が可能。


・既存スクリプトを編集する場合は、「スクリプト読み込み」ボタンより読み込み可能。
   エラーが起こる可能性も十分あるため、不具合あっても許してください。


// 注意事項
指定した兵科名とArma内の「RollDescription」欄（日本語表示「ロール説明」欄）を一致させないと、意図通り動作しません。
大文字小文字まで合わせてください。
